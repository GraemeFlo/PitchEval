# PitchEval

A referee evaluation and match‑logging web app for US Soccer Referee Coaches.

**Live app:** https://YOUR-USERNAME.github.io/pitcheval/

Open the link on a phone or computer browser. On iPhone/iPad you can tap **Share → Add to Home Screen** to use it like an installed app. All data is stored locally on the device.

## Features
- Live match clock, score, and per-team foul tracking
- Event logging: fouls, cards, offsides, goals, penalty kicks
- Critical Match Incident (CMI) and Critical Referee Error (CRE) records
- Grade-based referee ratings (Grassroots → FIFA)
- AI-assisted coaching feedback
- Match Library (saved matches) and text-file export
